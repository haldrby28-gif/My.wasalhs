# My Wasalha

تطبيق توصيل طلبات متكامل مشابه لتطبيق طلبات.

## مكونات المشروع
- customer_app : تطبيق العميل
- driver_app : تطبيق المندوب
- restaurant_app : تطبيق المطعم
- admin_panel : لوحة الإدارة
- Firebase : إعدادات Firebase

## المميزات
- تسجيل الدخول برقم الهاتف
- تحديد الموقع بالخريطة
- تتبع الطلب لحظياً
- الدفع نقداً أو إلكترونياً
- إشعارات فورية
